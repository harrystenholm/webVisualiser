# webVisualiser
A W.I.P recreation of my desktop music visualiser application for web browsers, using only HTML, CSS and JavaScript. Listens to and processes system audio from your device to display volume and waveform data as a web based graphic.

Works with microphone input and wired speakers / headphones (must enable stereo mix in sound settings for last two). Limited functionality with digital output (wireless / HDMI) due to system audio processing.
